title: Install GO in Linux Mint ( Debian Base )

---

Download golang at [link](https://golang.org/dl/) <br>
extract file to 

`/usr/local/go`

set ENVIRONMENT VARIABLE `GOROOT` to golang instalation path `/user/local/go` <br>
set ENVIRONMENT VARIABLE `GOPATH` to your golang work path <br>

run <br><br>
`$ go version`

<br>
See more [at](https://golang.org/doc/install)
