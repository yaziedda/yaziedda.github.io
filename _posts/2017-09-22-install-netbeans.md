Bahan yang di perlukan : 
* JDK ( Java Development Kit )
* Netbeans [download](https://netbeans.org/downloads/start.html?platform=windows&lang=en&option=all)

JDK adalah singkatan dari Java Development Kit yaitu software yang digunakan untuk membangun aplikasi-aplikasi java. Tanpa JDK kita tidak akan bisa membangun atau membuat berbagai macam aplikasi java.

1. Download JDK ( Java Development Kit ) at [link](http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html) <br><br>
![Alt text](/images/step1.PNG?raw=true "Step 1")
<b>Accept License Agreement, and download for your Windows version ( 32/64 bit ) x86 means 32 bit<br><br><br></b>
Untuk instalasi JDK, lihat tutorial di link berikut : <br>
[http://himatifubp.or.id/cara-mengisntall-java-development-kit-jdk-pada-windows/](http://himatifubp.or.id/cara-mengisntall-java-development-kit-jdk-pada-windows/)

### Instalasi Netbeans
Download at [link](https://netbeans.org/downloads/start.html?platform=windows&lang=en&option=all)
1. Buka file instalasi netbeans yang sudah di download tadi<br><br>
![Alt text](/images/step2.png?raw=true "Step 2")<br><br>
2. Klik Next<br><br>
![Alt text](/images/step3.png?raw=true "Step 3")<br><br>
3. Accept License Agreement dan Klik NEXT<br><br>
![Alt text](/images/step4.png?raw=true "Step 4")<br><br>
3. Pemilihan folder instalasi<br><br>
- Kolom pertama adalah dimana kita mau menyimpan folder instalasi Netbeans kita ( Lebih baik tidak di ubah )
- Kolom kedua adalah dimana tempat folder instalasi JDK yang kita install tadi<br><br>
- Bila sudah OK klik NEXT
![Alt text](/images/step5.PNG?raw=true "Step 5")<br><br>
4. Pemilihan folder instalasi glassfish<br><br>
Glashfish adalah webserver yang bisa di gunakan oleh java, namun sepertinya kita belum terlalu menggunakannya. Klik NEXT saja.<br><br>
![Alt text](/images/step6.PNG?raw=true "Step 6")<br><br>
5. Summary dari step step sebelumnya.<br><br>
- Centang check for updates bila ingin mendapatkan pemberitahuan tentang pembaharuan software Netbeans.
- Klik NEXT<br><br>
![Alt text](/images/step7.PNG?raw=true "Step 7")<br><br>
6. Instalasi sedang berjalan.<br><br>
Tunggu sampai selesai...<br><br>
![Alt text](/images/step8.PNG?raw=true "Step 8")<br><br>
7. Instalasi berhasil.<br><br>
Klik finish<br><br>
![Alt text](/images/step9.PNG?raw=true "Step 9")<br><br>
8. Buka Netbeans.<br><br>
![Alt text](/images/step10.png?raw=true "Step 10")
![Alt text](/images/step11.PNG?raw=true "Step 11")
