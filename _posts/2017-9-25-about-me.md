### Sekilas tentang saya
Nama saya Yazied Dhiya Uddien Abdurrahman atau biasa di panggil Yazid. Alasan kenapa saya masuk Teknik Informatika karena saya suka sekali dengan teknologi IPTEK apalagi di dunia Software Development. Menurut saya ini keren dan menantang, disamping itu dunia ini penuh dengan ke unik-an, bisa jadi tidak semua orang menyukai ini dan dalam teknisnya pun macam-macam rupanya dan sangat luas untuk di pelajari. Selain sangat menyukai dunia Software Development saya pun saya menyukai olahraga, lebih tepatnya di dunia Sepak Bola. Bermain hingga menonton saya sangat menyukainya. 

Pengalaman saya di bidang IT lebih banya di Software Development, hampir lebih 3 tahun dari saya duduk di bangku sekolah SMK saya sudah menyukainya. Ya ini sesuai dengan alasan saya mengapa menyukainya sampai saat ini yaitu banyak keunikan dan tantangan serta sangat luas untuk di pelajari. Dalam Software Development bahasa yang saya sangat sukai adalah java, apalagi java untuk membuat aplikasi Android hampir 2 tahun belakangan ini saya menekuninya dalam segi pekerjaan, riset atau hanya sekedar update teknologi terbaru di dalamnya.
